# დენი — საქართველოს ენერგიის მეურვეები
## გეგმის შეჯამება ქართულად · v2 · 11 ივლისი, 2026 (აშენებამდე ბოლო დოკუმენტი)

---

## რას ვაშენებთ

**„დენი"** — რიგებზე დაფუძნებული (turn-based) სტრატეგიული თამაში, mobile-first, დაინსტალირებადი PWA. მოთამაშე არის მეწარმე, რომელიც ₾10,000-ით იწყებს, ირჩევს საქართველოს რეგიონს და ეტაპობრივად ხდება ქვეყნის ენერგეტიკული მომავლის შემქმნელი.

**ჟანრი:** სტრატეგია/ტაიკუნი + სერიოზული თამაში (edutainment). შედარებები: Terra Nil (ტონი), Islanders (მინიმალისტური 3D დიორამა), Frostpunk (მორალური წნეხი).

## სამი აქტი (თამაშის ისტორია)

1. **აანთე შენი რეგიონი** — დააკმაყოფილე მოთხოვნა სტაბილურად. ისწავლი: სეზონები, ნდობა, ბლექაუთები, გაზის „ყავარჯენი".
2. **გაანათე საქართველო** — მეორე რეგიონი, გადამცემი ხაზი, შენახვა (ბატარეა, საქაჩი ჰესი). ისწავლი: რატომ არის საჭირო მიქსი და შენახვა.
3. **ევროპის მწვანე ბატარეა** — შავი ზღვის კაბელის წილის ყიდვა და ევროკავშირთან 4-კვარტლიანი საექსპორტო კონტრაქტის შესრულება. ეს **რეალური პროექტია**: საქართველო–რუმინეთი, 1,155 კმ, ~$2.3 მლრდ, ინვესტიციის საბოლოო გადაწყვეტილება — 2026 წლის აპრილი, სამი თვის წინ!

სრული თამაში: 12–18 წუთი. ჟიურისთვის: `?act=3` — პირდაპირ შუა თამაშში გადახტომა.

## ყოველი სვლის დილემა (რატომ არ არის მოსაწყენი)

ყოველ კვარტალში მოთხოვნა ან დაფარულია, ან — არა:

- **დეფიციტი?** → ① დაწვი **იმპორტირებული გაზი** (−ფული, +CO₂, +დამოკიდებულების მაჩვენებელი) ან ② **ბლექაუთი** (−ნდობა, −კეთილდღეობა, დიორამა რეალურად ბნელდება).
- **ჭარბი ენერგია?** → ③ **შეინახე** ან ④ **გაყიდე/ექსპორტი** (მაგრამ ზედმეტი ვალდებულება = ზამთარში შენივე სოფლები ბნელდება).

ეს ზუსტად ის დილემაა, რომლის წინაშეც რეალური საქართველო დგას: ~79% ჰიდრო / ~20% გაზი, გაზი კი ~100% იმპორტია (84% აზერბაიჯანიდან).

## ნამახვანის მექანიკა (თამაშის გული — არ იჭრება)

დიდი ჰესის აშენება შეუძლებელია საზოგადოების ნდობის (0–100) გარეშე: სოფლის კრებები, გამჭვირვალე გარემოსდაცვითი შეფასება, შემოსავლის გაზიარება. პირველად ჰესის ბარათის გახსნისას — სცენარული ინტერსტიციალი რიონის ხეობის 2020–21 წლების პროტესტებზე და არჩევანი: „ჩუმად გაატანე ნებართვები" (60% — პროტესტი, მშენებლობა იყინება, ინვესტიციის 30% იწვება) თუ „გააკეთე სწორად". იმერეთსა და რაჭაში ჰიდროპროექტებს +10 დამატებითი ნდობა სჭირდება — ნაიარევი მონაცემებშია ჩაშენებული.

## 7 რეგიონი

კახეთი (მზე) · ქართლი (ქარი — გორის პრეცედენტი) · ჯავახეთი (ქარი, მკაცრი ზამთარი) · იმერეთი და რაჭა (ჰიდრო + ნამახვანის ნაიარევი) · აჭარა და სამეგრელო (სანაპირო: ოფშორული ქარი + კაბელის მიღების წერტილი; სამეგრელო = „მზის ხაფანგი" — ღრუბლიანი რეგიონი სჯის დაუფიქრებელ კოპირებას).

## ტექნოლოგია (v2.1)

- **Vite + React 19 + TypeScript + Three.js + zustand + vite-plugin-pwa + Dexie (IndexedDB) + Clerk + Convex.** Next.js — არა: Clerk-ს აქვს ოფიციალური SPA SDK (`@clerk/clerk-react`), სერვერი საერთოდ არ გვჭირდება (Convex — მართული backend, ნული მოვლა), PWA/offline კი Vite-ში ბევრად უფრო მომწიფებულია.
- **Local-first შენახვა:** სტუმრის რეჟიმი = ნაგულისხმევი, თამაში მყისვე იწყება; ყოველი სვლის შემდეგ ავტოშენახვა IndexedDB-ში; შესვლისას (Clerk) ლოკალური პროგრესი ღრუბელში (Convex) აიტვირთება და მოწყობილობებს შორის სინქრონდება. ლიდერბორდი — realtime.
- **Three.js მხოლოდ დიორამას ხატავს; მთელი UI — React/DOM**, mobile-first, ცერით მართვადი. 3D რჩება: turn-based თამაში = render-on-demand (გადახურება/throttling გვერდს გვივლის), pixelRatio ≤2, ჩრდილები მობილურზე გამორთული, context-loss ჰენდლერი; გეგმა B (SVG) კვლავ ძალაშია.
- **PWA**: ჟიური QR კოდით ტელეფონზე დააინსტალირებს და ოფლაინშიც იმუშავებს.
- **სუფთა ძრავი** (DOM-ის, დროისა და შემთხვევითობის გარეშე, seeded RNG) + `sim.ts` — ბოტები თამაშობენ 500+ პარტიას და ამოწმებენ: ბალანსი მუშაობს, გაკოტრება/NaN/გაუთავებელი თამაში შეუძლებელია, „გაზზე დამჯდარი" ბოტი აგებს, ნდობის გამომტოვებელი — პროტესტს იღებს.
- **გეგმა B**: თუ 3D 2 საათში არ დადგა — SVG პანორამა იმავე ინტერფეისით. თამაში WebGL-ის მძევალი ვერ იქნება.

## ვიზუალი და ასეთები

დაბალპოლიგონიანი 3D დიორამა, თბილი ქართული პალიტრა, დღე-ღამის ციკლი. **ასეთების მილსადენი:** CC0 GLB მოდელები (poly.pizza, Kenney, Quaternius — უფასო, ატრიბუციის გარეშე) → ტექსტურების მოხსნა და ერთიანი toon მასალა კოდში (ერთიანი სტილი) → პროცედურული რელიეფი 7 რეგიონისთვის → Higgsfield AI მხოლოდ 2D არტისთვის (სათაური, რეგიონის ბარათები, ინტერსტიციალი, ხატულა). AI-ს 3D მოდელებს არ ვენდობით დემოზე.

## ეტაპები (ღამის ~12 საათი)

M0 კარკასი (0.5სთ) → M1 ძრავი + სიმულაცია მწვანეა (2სთ) → M2 სრული ციკლი DOM UI-ში (1.5სთ) → M3 დიორამა (2სთ, მკაცრი ლიმიტი) → M4 ივენთები + ნამახვანი (1.5სთ) → M5 II–III აქტები (1.5სთ) → M6 სილამაზე + ქართული (1.5სთ) → M7 ბალანსი, ტესტი ტელეფონზე, დეპლოი, QR (1სთ).

**წესი:** ყოველი ეტაპის ბოლოს თამაში მუშაობს. თუ დრო არ ჰყოფნის, იჭრება: ლიდერბორდი → სამეგრელო → ოფშორი → მეორე მყიდველი. **არასდროს იჭრება:** ნამახვანის ინტერსტიციალი, ბლექაუთის ჩაბნელება, ჟიურის რეჟიმი, ქართული ენა.

## ჟიურის პიჩი (90 წამი)

1. „საქართველო გაზის 100%-ს იმპორტით იღებს და ყველას გვახსოვს ნამახვანი." (კაუჭი)
2. ცოცხლად: ზამთრის დეფიციტი → არჩევანი: გაზი თუ სიბნელე. (დილემა)
3. ჰესის ნაჩქარევი მშენებლობა → პროტესტი. სწორად → ნდობა → ჰესი. (გაკვეთილი)
4. ფინალი: სინათლის რკალი საქართველოდან რუმინეთისკენ — „რეალურმა კაბელმა ინვესტიციის გადაწყვეტილება სამი თვის წინ მიიღო." (ვაუ)
5. კომერციალიზაცია: სკოლების ლიცენზიები, GNERC/დონორების კამპანიები, „ქვეყნის პაკეტები" (ძრავი მონაცემებზეა აგებული — სომხეთის/მოლდოვის ვერსია = მონაცემთა ფაილი), roadmap: co-op, დიპლომატია, სცენარები. QR — „დააინსტალირეთ ახლავე."
